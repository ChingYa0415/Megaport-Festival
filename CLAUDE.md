# 專案指引

## 語言規範
- 所有回覆一律使用**繁體中文（台灣正體）**
- 禁止使用中國大陸用語，一律使用台灣慣用詞彙，例如：
  - 程式碼（非「代码」）、資料（非「数据」）、變數（非「变量」）
  - 伺服器（非「服务器」）、網路（非「网络」）、軟體（非「软件」）
  - 元件（非「组件」）、函式（非「函数」）、迴圈（非「循环」）
  - 陣列（非「数组」）、物件（非「对象」）、介面（非「接口」）
  - 回呼（非「回调」）、參數（非「参数」）、型別（非「类型」）
  - 部署（非「部署」但語境要台灣化）、檔案（非「文件」指 file 時）
  - 訊息（非「消息」）、記憶體（非「内存」）、硬碟（非「硬盘」）
- 程式碼中的註解與使用者可見字串也遵守此規範

## 技術棧
- React 18 + TypeScript
- Vite 建構工具
- Tailwind CSS v4
- Firebase（Firestore + Auth + Storage + Hosting）
- browser-image-compression

## 專案概述
大港開唱 2026 好友行程共享手機版網頁應用。使用者在時間表上標記想聽的表演，朋友間透過 Firestore 即時同步選擇。

## 常用指令
- `npm run dev` — 啟動開發伺服器
- `npm run build` — TypeScript 檢查 + 建構
- `npx tsc --noEmit` — 僅做型別檢查
